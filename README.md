## ✨ Core Engineering Features

* **💾 Persistent Local State:** Integrates directly with the browser's `localStorage` API to ensure your lists, tasks, and configurations survive browser refreshes.
* **🏷️ Dynamic Filtering Matrix:** Seamlessly filter tasks by status (All, Active, Completed) using responsive UI filters without page reloads.
* **🎯 Intuitive Swipe/Click Controls:** Fast task marking, editing, and single-click deletion triggers designed for quick mouse and keyboard workflows.
* **🎨 Dark Mode Optimized UI:** A handpicked, high-contrast dark color palette combined with subtle modern transitions to eliminate late-night developer eye strain.
* **📱 100% Responsive Grid:** Engineered using fluid CSS Flexbox and Grid layouts, scaling perfectly from mobile screen viewports up to wide-screen development monitors.
